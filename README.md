# Introduction 
Bookings Project, a tool used to book rooms and manage your bookings

# Getting Started
1. Install Dependecies
```
yarn
```
or
```
npm install
```

2. Run the Server
```
cd server
node server
```

3. Run the Project
```
yarn start
```
or
```
npm run start
```

4. Documentation of Components
```
npm run storybook
```
